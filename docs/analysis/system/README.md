- 編集元は `.asta`

## レビュー観点

- ユースケースが要求する振る舞いを網羅している
- Boundary・Control・Entityの責務が分離されている
- コラボレーション図のメッセージとControlの操作が対応している
- Controlから必要なEntityへの依存がある
- Entity間の関連と多重度がドメインクラス図と一致している
- 基本系列だけでなく代替・例外系列も扱える
