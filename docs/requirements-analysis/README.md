## Review Checklist

- 要求される機能と関係するアクタが網羅されている
- 抽象的な機能名だけでなく、具体的な振る舞いが書かれている
- ユースケース名が現在時制かつ能動態の動詞句になっている
- 事前条件、事後条件、基本系列、代替・例外系列が明確である
- システムへの入力とシステムからの応答が書かれている
