## Review Checklist

- ユースケースが要求する振る舞いを網羅している
- 必要なデータとその置き場所が明確である
- 相互作用、操作、多重度が対応している
- バウンダリ、コントロール、エンティティの役割が合っている
- 基本系列だけでなく代替・例外系列も扱っている
